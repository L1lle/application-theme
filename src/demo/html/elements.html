Responsive Card Decks

Responsive card-deck with flexible amount of maximum columns. Depending on the screen size the amount of available columns will decrease. Add one of these classes .card-deck-column-6, .card-deck-column-4, .card-deck-column-2 to the .card-deck. This function is supposed to be included later in Bootstrap v4.1 or v4.2.
