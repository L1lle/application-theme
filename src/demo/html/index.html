Lightweight App-Theme

A simple and developer friendly basic app theme.

Based on Bootstrap4. This is work in progress. It's not yet recommended to use it in production.

Getting startet

Install the template as npm package.

npm install [name] --save-dev

Integrate into build process

This module does not mange or include any depencency JavaScript libraries. The main purpose is to provide the important CSS files to setup the theme. You can use any build tool or libraries of your choice.
